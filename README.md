# behebes.AI

behebes.AI ist eine modulare, mandantenfähige Plattform für modernes Bürgermelden in der kommunalen Praxis: vom Eingang einer Meldung über intelligente Kategorisierung bis zur operativen Abarbeitung im Team.

Das System vereint Bürgerfrontend, Admin-Backend und eine mobile Ops-PWA in einer durchgängigen Architektur mit Fokus auf Datenschutz, Nachvollziehbarkeit, Updatefähigkeit und Alltagstauglichkeit in Verwaltungen.

Projektleitung und Umsetzung: **Dominik Tröster**, Digitalbeauftragter der Verbandsgemeinde Otterbach-Otterberg.  
Dieses Projekt ist aus der Praxis entstanden und wird kontinuierlich weiterentwickelt.
